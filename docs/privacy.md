# プライバシー説明

INIADPLUS は、INIAD MOOCs の閲覧画面を中心に動作する Chrome 拡張です。

## 収集しない情報

- 氏名、メールアドレス、学籍情報などの個人情報
- 閲覧履歴や行動ログ
- 位置情報
- 認証情報やパスワード
- 解析・広告・トラッキング用データ

## ブラウザ内にのみ保存する情報

- ユーザーが入力したメモ
- 時間割 JSON などの設定情報
- 画面表示に使う簡単なオンオフ設定

これらはブラウザの `localStorage` に保存され、開発者のサーバーには送信されません。

## ウェブサイトのコンテンツの取り扱い

- INIAD MOOCs の講義ページや、埋め込まれた Google スライドの表示内容を、PDF / 画像 保存機能の実行時にブラウザ内でのみ処理します
- この処理は、利用者が押した PDF / 画像 保存ボタンに応じて行われます
- 取得した内容は PDF / 画像 の生成と画面表示のためだけに使われ、開発者の外部サーバーへ送信されません

## 外部送信

INIADPLUS は、開発者が管理する外部サーバーへデータを送信しません。

PDF / 画像 保存機能では、表示中の Google スライドの内容をブラウザ内で印刷用ページに整形し、Chrome 標準の印刷エンジン（PDF）または DevTools プロトコル（画像）を経由して、利用者のローカル環境にのみファイルを保存します。

## 必要な権限について

`manifest.json` で次の Chrome 拡張権限を要求します。いずれもローカル動作に閉じており、外部送信を伴いません。

- `debugger`: 表示中のスライドタブに対し、Chrome 標準の印刷エンジン（`Page.printToPDF`）またはスクリーンショット機能（`Page.captureScreenshot`）を呼び出すためにのみ使用します
- `downloads`: 生成した PDF / PNG をユーザーのダウンロードフォルダに保存するためにのみ使用します
- `storage`: ダウンロードキューや進行中ジョブの状態をローカルに保持するためにのみ使用します
