# INIAD++ (PDF Download Fork)

![説明](img/discription.PNG)

東洋大学情報連携学部(INIAD)のオンライン教育システム [INIAD MOOCs](https://moocs.iniad.org/) とその周辺サイト群に様々な機能を追加する Google Chrome™ 拡張機能です。

> **このリポジトリについて**
> 本リポジトリは [akahoshi1421/INIAD-](https://github.com/akahoshi1421/INIAD-) のフォークで、**「講義スライドのダウンロードを HTML ではなく PDF で出力するように変更」** した改変版です。
> 原作の機能・ライセンスはそのまま継承しています。オリジナル作者様に敬意を表します。

## インストール方法

Chrome ウェブストアには公開していません。下記の手順でローカルに読み込んで使います。

### 1. このリポジトリを取得する

#### 方法A: Git を使う
```bash
git clone https://github.com/Kensuke-sam/iniad_plus.git
cd iniad_plus
git checkout feat/pdf-download
```

#### 方法B: ZIP をダウンロードする（Git を使わない場合）
1. GitHub のリポジトリページで **ブランチ切替メニュー → `feat/pdf-download`** を選ぶ
2. 緑の **`Code`** ボタン → **`Download ZIP`**
3. ダウンロードした ZIP を任意のフォルダに解凍（例: `~/iniad_plus`）

### 2. Chrome に読み込む

1. Chrome のアドレスバーに `chrome://extensions/` と入力して開く
2. 右上の **「デベロッパーモード」** を ON にする
3. 左上の **「パッケージ化されていない拡張機能を読み込む」** をクリック
4. ステップ 1 で取得したフォルダ（**`manifest.json` が直下にある階層**）を選択
5. 一覧に **INIAD++** が表示されれば完了

> ※ フォルダを削除・移動すると動作しなくなります。そのままの場所で置いておいてください。

## 使い方（PDF ダウンロード）

1. INIAD MOOCs の講義ページを開く
2. スライド下部に表示される **「スライドをPDFでダウンロードする」** ボタンをクリック
3. 新しいタブで Google スライドの公開ビューが開き、確認ダイアログ **「スライドをPDFでダウンロードしますか？」** が出る → **OK**
4. 全スライドの取り込みが完了すると、**Chrome の印刷ダイアログが自動で開く**
5. 「送信先」を **「PDFに保存」** に設定し、**「保存」** を押して任意の場所に保存

### 補足

- 印刷ダイアログが自動で開かなかった場合は、置き換わったページ上部の緑色のバーにある **「印刷ダイアログを開く」** ボタンを押してください。
- スライド枚数によっては取り込みに数秒〜数十秒かかります。完了までタブを閉じないでください。
- 生成した PDF は **A4 横 / 1 ページ 1 スライド** のレイアウトで保存されます。
- うまく動かないときはタブで `F12` → `Console` を開き、`[INIAD++ PDF]` で始まるログを確認してください。

## 更新・アンインストール

### 更新する場合
```bash
cd iniad_plus
git pull
```
その後、`chrome://extensions/` で INIAD++ カードの **再読み込み（⟳）** ボタンを押すと反映されます。

### アンインストール
`chrome://extensions/` の INIAD++ カードから **「削除」** を押してください。

## 機能一覧

* 直近の講義のお知らせ
* ace から時間割のダウンロード
* **講義スライドの PDF ダウンロード**（本フォークの変更点）
* トップページに表示されているコース一覧の自由な入れ替え
* 1 ページごとにメモ機能（複数可）
* メモのダウンロード
* メモを保存したページ一覧を表示するページ
* 外部リンク一覧
* ドライブを開くボタン
* 背景色の変更
* ヘッダおよびサイドバーの追従（アカウント設定ページから変更可能）
* ページ上部に戻るボタン
* 入力文字カウント
* スライド位置、大きさの操作
