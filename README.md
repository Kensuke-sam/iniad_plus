# INIAD++ (PDF Download Fork)

![説明](img/discription.PNG)

東洋大学情報連携学部(INIAD)のオンライン教育システム [INIAD MOOCs](https://moocs.iniad.org/) とその周辺サイト群に様々な機能を追加する Google Chrome™ 拡張機能です。

> **このリポジトリについて**
> 本リポジトリは [akahoshi1421/INIAD-](https://github.com/akahoshi1421/INIAD-) のフォークであり、**「スライドのダウンロードを HTML ではなく PDF で出力するように変更」** した改変版です。
> オリジナル作者様に敬意を表します。原作の機能・ライセンスはそのまま継承しています。

## インストール方法（開発者モードで読み込む）

Chrome ウェブストアには公開していないため、ローカルで「パッケージ化されていない拡張機能」として読み込みます。下記の手順は誰でも実行できます。

### 1. このリポジトリを取得する

#### 方法A: Git を使う
```bash
git clone https://github.com/Kensuke-sam/iniad_plus.git
cd iniad_plus
git checkout feat/pdf-download
```

#### 方法B: ZIP をダウンロードする（Git を使わない場合）
1. GitHub のリポジトリページで `feat/pdf-download` ブランチに切り替える
2. 緑の `Code` ボタン → `Download ZIP`
3. ダウンロードした ZIP を任意のフォルダに解凍（例: `~/Downloads/iniad_plus`）

### 2. Chrome に読み込む

1. Chrome のアドレスバーに `chrome://extensions/` と入力して開く
2. 画面右上の **「デベロッパーモード」** を ON にする
3. 左上の **「パッケージ化されていない拡張機能を読み込む」** をクリック
4. ステップ 1 で取得したフォルダ（`manifest.json` が直下にあるフォルダ）を選択
5. 一覧に **INIAD++** が表示されれば完了

> ※ フォルダの場所を後から移動・削除すると動作しなくなります。削除しない場所に置いてください。

### 3. 使い方（PDF ダウンロード）

1. INIAD MOOCs の講義ページを開く
2. スライド下部に表示される **「スライドをPDFでダウンロードする」** ボタンをクリック
3. 新しいタブで Google スライドの公開ビューが開き、確認ダイアログ **「スライドをPDFでダウンロードしますか？」** が出る
4. 「OK」を押すと全スライドを自動で 1 枚ずつ取得し、PDF として保存される

> ダウンロード中は新タブのページタイトルに `[PDF生成中 N/M]` のように進捗が表示されます。完了するまでタブを閉じないでください。
> うまくいかない場合は、そのタブで `F12` → `Console` を開いて `[INIAD++ PDF]` で始まるログを確認してください。

### 4. 更新する場合

```bash
cd iniad_plus
git pull
```

その後、`chrome://extensions/` のこの拡張機能のカードにある **再読み込み（⟳）** ボタンを押すと反映されます。

### 5. アンインストール

`chrome://extensions/` の該当カードから「削除」を押してください。

## 機能

* 直近の講義のお知らせ
* ace から時間割のダウンロード
* **講義スライドの PDF ダウンロード**（本フォークの変更点）
* トップページに表示されているコース一覧の自由な入れ替え
* 1 ページごとにメモ機能（複数可）
* メモのダウンロード
* メモを保存したページ一覧を表示するページ
* 外部リンク一覧
* ドライブを開くボタン
* 背景色の変更
* ヘッダおよびサイドバーの追従
  （上記 2 つはアカウント設定ページから変更できます）
* ページ上部に戻るボタン
* 入力文字カウント
* スライド位置、大きさの操作
